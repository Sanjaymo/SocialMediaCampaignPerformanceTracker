### Data Preparation & ETL

**1. Data Import & Cleaning**
```dax
// Power Query M Code
let
    Source = Excel.Workbook(File.Contents("marketing_data.xlsx")),
    CleanedData = Table.TransformColumnTypes(Source,{
        {"TV", Currency.Type},
        {"Billboards", Currency.Type},
        {"Google_Ads", Currency.Type},
        {"Social_Media", Currency.Type},
        {"Influencer_Marketing", Currency.Type},
        {"Affiliate_Marketing", Currency.Type},
        {"Product_Sold", Int64.Type}
    })
in
    CleanedData
```

**2. Data Transformation**
- Standardized currency formats across all spend columns
- Handled missing values using forward-fill method
- Created date hierarchy for time-series analysis
- Normalized product sold figures for consistency

### DAX Measures & Calculations

**Key DAX Measures:**

```dax
// Total Marketing Spend
Total Marketing Spend = 
    SUM(Marketing[TV]) + 
    SUM(Marketing[Billboards]) + 
    SUM(Marketing[Google_Ads]) + 
    SUM(Marketing[Social_Media]) + 
    SUM(Marketing[Influencer_Marketing]) + 
    SUM(Marketing[Affiliate_Marketing])

// Total Products Sold
Total Products Sold = SUM(Marketing[Product_Sold])

// Cost per Product (Key Efficiency Metric)
Cost per Product = 
    DIVIDE(
        [Total Marketing Spend],
        [Total Products Sold],
        0
    )

// Google Ads Percentage
Google Ads % = 
    DIVIDE(
        SUM(Marketing[Google_Ads]),
        [Total Marketing Spend],
        0
    )

// Channel-Specific ROI
Channel ROI = 
    DIVIDE(
        [Total Products Sold] * [Average Product Price],
        [Total Marketing Spend],
        0
    ) - 1

// Marketing Efficiency Score
Marketing Efficiency = 
    DIVIDE(
        [Total Products Sold],
        [Total Marketing Spend] / 1000,
        0
    )
```
